## 部署指导

项目是百度首页的高仿版，可以起到部分的迷惑作用。

项目代码较为精简且包含较为完整的注释，仅引入了jQuery和clipboard插件，阅读代码或修改应该不是很难。

对于带有参数的链接会自动运行动画，当用户点击输入框或者搜索按钮时，动画会被打断并显示为初始状态。

你可以直接在自己的服务器中部署该项目中的源代码，以静态页面的方式运行即可。
